Activity: Reduce the size of the browser window so that the main navigation menu transforms into a drop-down menu button. Click the button twice. Once to show the menu and a second time to hide it. Then read the description below to better understand how it works.

This web page represents a minimal Bootstrap template with its signature responsive main navigation menu.

The code below illustrates what "mobile first, desktop friendly" really means and how Bootstrap implements it.

The drop-down button (navbar-toggle) is shown by default. The collapsible main menu (.collapse) is hidden by default and JavaScript is used to implement the drop-down functionality.

The @media rule is used to override the default mobile presentations. CSS is used to hide the drop-down button (navbar-toggle) and to re-position the main menu (.collapse) as a horizontal menu instead of a drop-down.

        
<div class="navbar-header">
  <!-- Drop Down Button -->         
  <button type="button" class="navbar-toggle">
    <span class="icon-bar"></span>
    <span class="icon-bar"></span>
    <span class="icon-bar"></span>
  </button>
  <a class="navbar-brand" href="#">Website Name</a>
</div>

<!-- 
Main Menu that can be displayed three ways:
1. As drop down for devices with display width less than 768
2. As a horizontal bar menu for display width greater than 768
3. As a vertical list for devices that might not fully support CSS and JavaScript
-->     
<div class="collapse navbar-collapse">
  <ul class="nav navbar-nav">
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</div>
