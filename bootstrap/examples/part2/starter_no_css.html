Bootstrap without CSS

Activity: This represents what a Bootstrap template would look like without CSS. Read the information below and fix this page to restore the styles.

Bootstrap's stylesheet is not just about style, it is an essential part of how responsiveness is implemented.

A common misconception is that good front-end frameworks can gracefully respond to web browsers that might not support CSS or JavaScript at all. This would be like expecting a Java program to gracefully respond to a computer that does not have a Java virtual machine; It is an unreasonable expectation.

This web page is located at examples/part2/starter_no_css.html

Bootstrap's JavaScript is located in the folder examples/css/

Below is the essential code that must be included for Bootstrap to function. This code is missing from this web page's head tag.

        
    
    <link href="../css/bootstrap.css" rel="stylesheet">
