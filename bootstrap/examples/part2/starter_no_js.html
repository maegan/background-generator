Bootstrap without JavaScript

Activity: Reduce the size of the browser window so that the main menu transforms into a drop-down menu. Try clicking the drop down menu. This web page represents how Bootstrap would behave without JavaScript. Read the information below and fix this page so the drop-down menu will work.

Bootstrap requires JavaScript and the jQuery library and cannot adaptively respond to the lack of JavaScript since many aspects of Bootstrap's responsiveness are implemented with JavaScript.

This web page is located at examples/part2/starter_no_js.html

Bootstrap's JavaScript is located in the folder examples/js/

Below is the essential code that must be included for Bootstrap to function. This code should be placed immediately above the body tag.

        
           
    <!-- Bootstrap core JavaScript
          Placed at the end of the document immediately
          above the body tag so the page load faster
    ================================================== -->
    <script src="../js/jquery.min.js"></script>
    <script src="../js/bootstrap.min.js"></script>
