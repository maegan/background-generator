Bootstrap Grids: Advanced

Activity: Bootstrap's four tiers of classes: xs (phones), sm (tablets), md (desktops), and lg (larger desktops) can be combined to create more dynamic and flexible layouts. Study the three examples below and then create a new layout that has 4 columns for desktop (md) display but 2 rows and 2 columns for tablet (sm) display. You can modify one of the existing examples on this web page (examples/part3/grid3.html)

Example 1

Narrow right column gets proportionally wider as the browser width gets smaller.

.col-xs-8 .col-sm-9 .col-md-10
.col-xs-4 .col-sm-3 .col-md-2

Example 2

Two columns until the browser width gets very small and then the right column becomes a half-sized row.

.col-xs-12 .col-sm-8
.col-xs-6 .col-sm-4

Example 3

Three columns until the browser width gets small and then the right column becomes a new row that continues to match the width of the middle row.

.col-sm-4 .col-md-2
.col-sm-8 .col-md-5
.col-sm-8 .col-md-5
