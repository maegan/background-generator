Bootstrap Grids: Applying the Concept

Activity: Create a layout on this web page with the following properties:

  • For desktop (md) display, the top of the page should have two equal-width columns.
  • And, the bottom of the page should have three columns: a narrow left- and right-side column with a wider center column.
  • When the browser width becomes small (sm), the two top columns should stack as rows.
  • And, the bottom right-side column should move down to a new row so the wider center column can occupy its space.

Activity: Instead of modifying an existing layout, try to create a layout from scratch right below these instructions. This page is located at examples/part3/grid4.html. Keep in mind that:

  • Rows must be placed within a <div class="container"> which is already part of this document.
  • Use rows <div class="row"> to create horizontal groups of columns.
  • Content should be placed within columns <div class="col-md-6">, and only columns may be immediate children of rows.
