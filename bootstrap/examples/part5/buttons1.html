Activity: The left column below includes a lengthy main menu that might be hard to use on a small touchscreen device. First, the items are close together, which requires a precision touch. Second, the menu is lengthy, which requires scrolling, and scrolling can sometimes trigger an accidental touch.

The right column below shows a fragment of the source code of the lengthy menu and below it is the Bootstrap code to make this menu mobile friendly. Change the code directly on this web page (examples/part5/buttons1.html).
